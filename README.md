
# 🧰 Universal Script Toolkit

Welcome to the **Universal Script Toolkit** — a curated collection of helper, project, and generic scripts designed to streamline tasks across **Windows**, **macOS**, **Windows Server**, and **Microsoft 365/Microsoft Graph** environments.

---

## 📁 Repository Structure

Scripts are organized by platform and purpose:

Each folder contains scripts with descriptive filenames and inline documentation where applicable.

---

## 🛠️ Supported Platforms

- **Windows 10/11**
- **Windows Server 2016+**
- **macOS Monterey and later**
- **Microsoft 365 / Microsoft Graph API**

---

## ✨ Features

- Automation helpers for system administration
- Microsoft Graph API utilities
- PowerShell, Bash, and cross-platform scripts
- Reusable functions and templates
- Community-driven improvements

---

## 📬 Requesting a New Script

To request a new script:

1. Open an issue
2. Use the **Script Request** template
3. Provide:
   - Platform (e.g., Windows, macOS)
   - Purpose or problem to solve
   - Any relevant context or examples

---

## 🧪 Suggesting Improvements

To suggest changes or improvements:

1. Fork the repository
2. Create a new branch: `feature/your-change-name`
3. Make your changes
4. Submit a Pull Request (PR) with:
   - A clear description of the change
   - A link to the related issue (if applicable)
   - Testing notes or screenshots (if relevant)

---

## 🔁 Pull Request Process

We welcome contributions! Here's how to get started:

1. Ensure your code follows the style and structure of existing scripts
2. Include comments and usage examples where helpful
3. Test your script on the relevant platform(s)
4. Submit a PR and tag relevant reviewers

> **Note:** All scripts are reviewed for clarity, security, and platform compatibility before merging.

---

## 🧾 Contribution Guidelines

- Keep scripts modular and reusable
- Avoid hardcoding sensitive data
- Use environment variables or config files where appropriate
- Document dependencies and prerequisites
- Prefer cross-platform compatibility when possible
